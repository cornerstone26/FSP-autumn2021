lab1test
========
Набор тестовых программ для лабораторной № 1 по ОСП. Для сборки - make all.

01. lab1test
------------
Программа для проверки заданной библиотеки на предмет её соответствия требованиям API плагинов.
Пример запуска:
 $ ./lab1test ./libavg.so

02. lab1call
------------
Программа для вызова из плагина функции проверки файла. Дополнительно выводит информацию о плагине.
Пример запуска:
 $ ./lab1call ./libavg.so --entropy 0.5 --offset-from 100 test.txt

03. libavg.so
-------------
Пример плагина для лабораторной № 1. Позволяет подсчитать энтропию файла или отдельного участка файла. Функция plugin_process_file() возвращает 0, если энтропия меньше заданного значения, и 1 в противном случае.
Поддерживаемые опции:
 --entropy v
    Проверка, что энтропия меньше v, v = [0.0, 1.0)
 --offset-from off_from
    Начать с байта со смещением off_from
 --offset-to off_to
    Закончить байтом со смещением off_to (не включая его)
