01. ex4.c
Пример серверной программы, использующей для мультиплексирования ввода-вывода вызов select(). Для подключения к серверу можно использовать telnet или netcat (nc):
> telnet 127.0.0.1 5555
> nc 127.0.0.1 5555

02. ex5.c
Пример ожидания ввода с клавиатуры. Если пользователь не успевает что-то напечатать за отведенные 5 секунд, то выбирается ответ по умолчанию.

03. ex6.c
Пример серверной программы, использующей для мультиплексирования ввода-вывода вызов poll(). Для подключения к серверу можно использовать telnet или netcat (nc):
> telnet 127.0.0.1 5555
> nc 127.0.0.1 5555
