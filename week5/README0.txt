01_signals
==========

01. ex1.c
Использование функции signal() для установки обработчика сигнала. Функцию signal() использовать не рекомендуется!

02. ex2.c
В настоящее время для установки обработчиков сигналов следует использовать системный вызов sigaction().

03. ex3_r.c
Пример получения сигнала реального времени и проверки значения аргумента сигнала.

04. ex3_s.c
Пример отправки сигнала реального времени с заданным аргументом (целым числом).

05. ex4.c
Пример генерации SIGSEGV при доступе по некорректному указателю.

02_sockets
==========

01. ex1c.c, ex1s.c
Пример серверной и клиентской программ, демонстрирующих подключение с помощью потоковых сокетов (TCP). Сервер возвращает строку из случайных символов.

02. ex2c.c, ex2s.c
Пример серверной и клиентской программ, демонстрирующих подключение с помощью потоковых сокетов (TCP). Сервер и клиент обмениваются случайными сообщениями для имитации чата.

03. ex3c.c, ex3s.c
Пример серверной и клиентской программ, демонстрирующих подключение с помощью дейтаграммных сокетов (UDP). Сервер и клиент обмениваются случайными сообщениями для имитации чата.
