01_c
====
Примеры для повторения C.

01. hello.c, hello_a.c
Специфическая платформозависимая версия хеллоуворлда без *функции* main.
`nm hello` позволяет увидеть символы в коде, среди них есть _start и main. Важно само наличие этих символов. Управление передается сначала на _start, затем на адрес main. Фактически получается, что управление передается на первый байт массива

02. hello4real.c
Обычный переносимый хеллоуворлд.

03.war.c
Пример программы с ссылкой на внешнюю функцию. Компилируется (`gcc -c war.c`), но не собирается (`gcc -o war war.c). Если добавить тело функции, то ждет ввода от пользователя из стандартного потока ввода.

04. qsort.c
Пример использования функции для сортировки массивов из стандартной библиотеки C.

05. qsortlambda.c
Еще один пример использования функции qsort(). В данном случае показана возможность определения функции сравнения элементов (функции обратного вызова для qsort) "как бы" без имени (на самом деле имя есть) с помощью макросов и расширений языка C, поддерживаемых gcc (вложенные функции [nested functions] и операторные выражения [statement expressions]).

06. search1.c
Поиск в многомерном массиве. Размерность фиксированная - 4 измерения. В программе есть серьзный баг, который не приводит к ошибкам компиляции. Также отсутствует обработка ошибок выделения памяти.

07. search2.c
Поиск в многомерном массиве произвольной размерности.

08. search3.c
То же самое, что и в search2.c, но разбитое на несколько единиц трансляции.

Пример сборки проекта с помощью make.

03_syscalls
===========

01. ex01_syscall.S, ex02_int80.S
Примеры выполнения системных вызовов на языке ассемблера nasm.

02. ex03_libc.S
Пример вызова функции из стандартной библиотеки на языке ассемблера nasm.

03. ex04_inline.c
Пример использования встроенного ассемблера gcc.

04. ex05_noglibc.c
Выполнение системного вызова совместимым с 32-х разрядным кодом способом без использования стандартной библиотеки.

05. ex06_diet.c
Использование стандартной библиотеки dietlibc (https://en.wikipedia.org/wiki/Dietlibc)

06. mystdlib
Пример создания зародыша собственной стандартной библиотеки (статическая линковка с исполняемым файлом).
