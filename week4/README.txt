01_create
=========
Примеры на создание потоков с помощью библиотеки pthread.

01. ex1.c
Модификация переменной в одном процессе не приводит к изменениям в другом. При модификации работает механизм COW (Copy on Write).

02. ex2.c
Модификация переменной в разных потоках одного процесса. Потоки видят изменения, сделанные друг другом, т.к. работают в одном адресном пространстве.

03. ex3.c
Использование атрибутов потока для того, чтобы создать поток с заданным размером стека. Только основной поток может менять размер стека динамически (см. man pthread_setstacksize). У остальных потоков он фиксирован - при создании можно задать размер.

04. ex4.c
Пример использования функций pthread_cancel() и pthread_testcancel().

05. ex_tgid.c
Получение идентификатора потока с помощью системного вызова gettid().

02_sync
=======

01. ex1.c
Несколько потоков изменяют счетчик без синхронизации.

02. ex1a.c
По сравнению с предыдущим примером добавлена синхронизация с помощью атомарных операций.

03. ex1mut.c
Синхронизация потоков, увеличивающих счетчик, с помощью мьютекса.

04. ex1s.c
Синхронизация потоков, увеличивающих счетчик, с помощью циклической блокировки.

05. ex2.c
Использование условных переменных (conditional variables) для синхронизации "производящих" и "потребляющих" потоков.

06. ex3.c
Использование блокировки чтения-записи для синхронизации работы "читателей" и "писателя".
